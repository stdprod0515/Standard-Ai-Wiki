# 챗봇 종류 및 특징

## 챗봇 개발을 위한 기초 배경 지식

### 1. 챗봇이란 무엇인가?

*   **정의:** 사람과 문자 또는 음성으로 대화하는 컴퓨터 프로그램.
*   **작동 원리:**
    *   **NLU (자연어 이해):** 사용자의 텍스트 의미, 의도 파악.
    *   **NLG (자연어 생성):** 자연스러운 응답 생성.
    *   **대화 관리:** 대화 맥락 유지 및 이어가기.

### 2. AI 에이전트 및 LLM (Large Language Model)의 역할

*   **AI 에이전트:** 특정 목표 달성을 위해 환경을 인지, 계획, 실행하는 시스템. 챗봇은 대화 환경에서 작동하는 AI 에이전트.
*   **LLM:** 챗봇의 핵심 기술. 방대한 텍스트 학습을 통해 언어 이해, 생성, 추론 능력을 갖춤. Gemma 4와 같은 모델이 해당.

### 3. 프로그래밍 기초

*   **프로그래밍 언어:** Python (AI 및 웹 개발에 주로 사용).
*   **Python 기초:** 변수, 데이터 타입, 조건문, 반복문, 함수 등.
*   **API (Application Programming Interface):** 프로그램 간 통신 및 데이터 교환 창구. LLM 사용, OpenClaw 프레임워크 활용 시 필요.

### 4. Anti-Gravity 플랫폼과 Gemma 4 모델

*   **Anti-Gravity:** AI 에이전트 중심 개발 환경. AI 에이전트에게 역할을 부여하고 지시하는 방식으로 개발.
*   **Gemma 4:** 챗봇의 '뇌' 역할. 텍스트 이해, 생성, 프로그램 개발 아이디어 제공 등에 활용.

### 5. 챗봇 개발 단계

1.  **목표 설정:** 챗봇 종류 및 기능 정의.
2.  **환경 설정:** Anti-Gravity, Gemma 4 로컬 구동 환경 설정.
3.  **AI 에이전트 설계:** 역할 부여, 지식/규칙 설정.
4.  **데이터 준비 (필요시):** 학습 데이터 가공.
5.  **개발 및 테스트:** Anti-Gravity에서 에이전트 지시, 기능 개발, 테스트.
6.  **배포 및 개선:** 실제 사용 및 피드백 기반 개선.

## 처음 개발하기 좋은 챗봇 종류 및 수요 높은 주제

### 처음 개발하기 좋은 챗봇 종류

*   **규칙 기반 또는 메뉴 기반 챗봇:** 미리 정의된 규칙, 키워드, 메뉴 옵션에 따라 작동. 대화 흐름 단순, 예측 가능. FAQ 답변, 간단한 정보 제공, 반복 문의 응대에 효과적.
*   **노코드/로우코드 플랫폼 활용:** 코딩 없이 시각적 인터페이스, 드래그 앤 드롭으로 챗봇 구축 (예: Botpress). Anti-Gravity 플랫폼 활용.
*   **단순한 흐름 유지:** 복잡한 분기 및 예외 처리 최소화.

### 수요 높은 챗봇 주제

*   **고객 서비스 및 FAQ:** 24시간 즉각 응답, 문의 처리, 문제 해결. (가장 일반적, 규칙 기반 시작 후 AI 통합 용이)
*   **전자상거래:** 제품 정보, 주문 추적, 고객 문의 자동화.
*   **정보 검색 및 Q&A:** 특정 주제에 대한 질문 답변, 정보 제공. (AI 학습/검색 능력 활용)
*   **콘텐츠 생성 보조:** 마케팅 문구, 글쓰기 보조, 아이디어 제안. (Gemma 4 텍스트 생성 능력 활용)
